  <behavior>
    get_category(entry: dict, fs=None) -> str
    Priority order: broken_symlink > symlink > directory > executable > archive > image > file

    Cases:
      {"islink": True, "destination": "/nonexistent/path"} with fs=None -> "broken_symlink"
        (os.path.exists fallback when fs is None; destination doesn't exist locally)
      {"islink": True, "destination": "/"} with fs=None -> "symlink"
        (/ always exists; valid symlink)
      {"islink": False, "mode": 0o040755} -> "directory"
        (stat.S_ISDIR checks the mode correctly)
      {"islink": False, "mode": 0o100755} -> "executable"
        (mode & 0o111 != 0 for user/group/other execute bits)
      {"islink": False, "mode": 0o100644, "name": "archive.tar.gz"} -> "archive"
      {"islink": False, "mode": 0o100644, "name": "photo.jpg"} -> "image"
      {"islink": False, "mode": 0o100644, "name": "README.md"} -> "file"
      {"islink": False, "mode": 0, "name": "unknown"} -> "file"
        (zero mode: default to file, not executable)
      {"islink": True, "destination": ""} with fs=None -> "symlink"
        (empty destination: treat as valid, per pitfall 1 — never misidentify non-symlinks)

    human_size(n: int | None) -> str
    Cases:
      None -> "    -"
      0 -> "0.0B"
      512 -> "512.0B"
      1024 -> "1.0K"
      1536 -> "1.5K"
      1048576 -> "1.0M"
      1073741824 -> "1.0G"

    format_mtime(mtime: float | None) -> str
    Cases:
      None -> "            " (12 spaces, matching column width)
      1741564800.0 -> strftime("%b %d %H:%M") of that timestamp (exact value depends on local TZ; test format shape not exact value)

    uid_to_name(uid: int) -> str
    Cases:
      os.getuid() -> current username (real call, not mocked)
      999999999 -> "999999999" (unknown UID falls back to str(uid))
      -1 -> "-1" (negative UIDs raise OverflowError on some platforms; fallback)
  </behavior>

    Create src/probefs/rendering/__init__.py (empty, just package marker).

    Create src/probefs/rendering/metadata.py:

    1. ARCHIVE_EXTS and IMAGE_EXTS as module-level frozensets (from research patterns).

    2. get_category(entry: dict, fs=None) -> str:
       - Gate ALL symlink logic on entry.get("islink", False) first (Pitfall 1 from research)
       - When islink=True: if destination is non-empty, check existence:
         - If fs is not None: call fs._fs.exists(destination)
         - Else: call os.path.exists(destination)
         - If not exists: return "broken_symlink"
       - Return "symlink" for valid symlinks
       - stat.S_ISDIR(mode) for directories
       - bool(mode & 0o111) for executables
       - os.path.splitext(name.split("/")[-1])[1].lower() for extension check
       - Use the ProbeFS instance (fs param) for broken symlink detection when provided,
         os.path.exists otherwise. This is the FAL boundary: widgets pass fs; standalone
         callers don't need to.

    3. human_size(n: int | None) -> str:
       - Return "    -" for None
       - Loop through ("B", "K", "M", "G", "T"), divide by 1024 until < 1024
       - Format: f"{n:.1f}{unit}"

    4. format_mtime(mtime: float | None) -> str:
       - Return 12 spaces for None (keeps column alignment when mtime missing)
       - datetime.datetime.fromtimestamp(mtime).strftime("%b %d %H:%M")

    5. uid_to_name(uid: int) -> str:
       - try: pwd.getpwuid(uid).pw_name
       - except (KeyError, OverflowError): return str(uid)

    No mocking needed for most cases; uid_to_name tests use real os.getuid() for
    the valid case and a hardcoded large integer for the fallback case.
