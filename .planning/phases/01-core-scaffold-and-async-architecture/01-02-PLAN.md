<feature>
  <name>FileManagerCore navigation state machine</name>
  <files>src/probefs/core/file_manager.py, tests/test_file_manager_core.py</files>
  <behavior>
    FileManagerCore manages navigation state. It does NOT perform filesystem I/O inside
    descend() or ascend() — those are pure path string operations. Workers on the screen
    call fs.ls() separately after a navigation event.

    State:
      cwd: str          — current working directory as absolute path string
      cursor_index: int — index of highlighted entry in the current pane (ListView.index)

    Methods:
      parent_path -> str         — PurePosixPath(cwd).parent as string; "/" at root
      descend(entry_name: str) -> str  — sets cwd = cwd/entry_name, cursor_index = 0, returns new cwd
      ascend() -> str            — sets cwd = parent_path, cursor_index = 0, returns new cwd

    Test cases:
      descend("/home/user", "projects") -> cwd == "/home/user/projects", cursor_index == 0
      descend("/home/user/projects", "probefs") -> cwd == "/home/user/projects/probefs"
      ascend("/home/user/projects") -> cwd == "/home/user", cursor_index == 0
      ascend("/home/user") -> cwd == "/home", cursor_index == 0
      ascend("/") -> cwd == "/", cursor_index == 0  (root does not go above /)
      parent_path("/home/user/projects") -> "/home/user"
      parent_path("/") -> "/"
      cursor_index resets to 0 on every navigation action
      ProbeFS instance is stored on self.fs (for worker access from MainScreen)
  </behavior>
  <implementation>
    Use PurePosixPath from pathlib for path arithmetic ONLY — no filesystem access.
    PurePosixPath does not hit the filesystem; it is pure string manipulation.
    ProbeFS is injected at __init__ and stored on self.fs — it is NOT called inside
    descend() or ascend(). The screen's @work(thread=True) worker calls self.app.screen.core.fs.ls()
    after navigation events.

    ```python
    from __future__ import annotations
    from pathlib import PurePosixPath
    from probefs.fs.probe_fs import ProbeFS

    class FileManagerCore:
        def __init__(self, fs: ProbeFS, start_path: str) -> None:
            self.fs = fs
            self.cwd: str = start_path
            self.cursor_index: int = 0

        @property
        def parent_path(self) -> str:
            parent = PurePosixPath(self.cwd).parent
            return str(parent)

        def descend(self, entry_name: str) -> str:
            self.cwd = str(PurePosixPath(self.cwd) / entry_name)
            self.cursor_index = 0
            return self.cwd

        def ascend(self) -> str:
            parent = self.parent_path
            if parent != self.cwd:  # at root, parent == cwd == "/"
                self.cwd = parent
            self.cursor_index = 0
            return self.cwd
    ```

  </implementation>
</feature>
