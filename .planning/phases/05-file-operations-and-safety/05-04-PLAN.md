  <how-to-verify>
Run `uv run probefs` and navigate to a directory with some test files. Then verify each operation:

**Setup (if needed):** Navigate to your home directory or any directory where you can safely create/delete test files.

**1. New file (n)**
- Press `n`
- Verify: InputDialog opens with empty input and cursor focused
- Type `test-probefs.txt` and press Enter
- Verify: Dialog closes, `test-probefs.txt` appears in the listing

**2. New directory (ctrl+n)**
- Press `ctrl+n`
- Verify: InputDialog opens with empty input
- Type `test-probefs-dir` and press Enter
- Verify: Dialog closes, `test-probefs-dir/` appears in the listing

**3. Rename (r)**
- Move cursor to `test-probefs.txt`
- Press `r`
- Verify: InputDialog opens pre-populated with `test-probefs.txt` and text is selected
- Clear the field and type `test-probefs-renamed.txt`, press Enter
- Verify: Dialog closes, listing shows `test-probefs-renamed.txt` (old name gone)

**4. Copy (y)**
- Move cursor to `test-probefs-renamed.txt`
- Press `y`
- Verify: InputDialog opens with a destination path pre-filled (current dir + filename)
- Append `-copy` before the `.txt` extension to make `test-probefs-renamed-copy.txt`
- Press Enter
- Verify: Dialog closes, both original and copy appear in listing

**5. Move (p)**
- Move cursor to `test-probefs-renamed-copy.txt`
- Press `p`
- Verify: InputDialog opens with destination pre-filled
- Change the filename portion to include `test-probefs-dir/moved.txt` (move into the new dir)
- Press Enter
- Verify: Dialog closes, `test-probefs-renamed-copy.txt` gone from listing; navigate into `test-probefs-dir/` to confirm `moved.txt` is there

**6. Delete (d)**
- Navigate back to the parent directory
- Move cursor to `test-probefs-renamed.txt`
- Press `d`
- Verify: ConfirmDialog opens asking to send to Trash
- Press Yes (or Tab to Yes then Enter)
- Verify: Dialog closes, `test-probefs-renamed.txt` gone from listing
- Open macOS Finder Trash (or OS equivalent) and confirm `test-probefs-renamed.txt` is there (not permanently deleted)

**7. Cancel behavior**
- Move cursor to any entry
- Press `d`
- When ConfirmDialog opens, press Escape or click Cancel
- Verify: Dialog closes, no operation performed, entry still in listing

**8. Directory operations**
- Press `d` on `test-probefs-dir/`
- Confirm: verify the directory (and `moved.txt` inside) goes to Trash

**Cleanup:** Empty macOS Trash when done testing if desired.
